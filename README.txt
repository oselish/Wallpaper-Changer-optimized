Перед первым запуском программы откройте файл config.txt и впишите четыре имени файлов: утренние, дневные, вечерние и ночные обои. Все эти файлы должны находится в папке Wallpapers в корневой папке программы.

Также перед первым запуском программы для файла WallaperChanger.exe (он может иметь другое имя, но не igcmdWin10.exe!!!) задайте запуск с правами администратора, чтобы программа могла запускаться при запуске системы. Для этого выберите нажмите правой кнопкой мыши по файлу -> Свойства -> Совместимость -> Запускать от имени администратора -> Применить -> ОК
